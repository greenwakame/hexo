### 投稿記事の確認
```
hexo server
```

<!-- more -->

### 下書き記事を作成

```
hexo new draft introduction
```

### 下書き記事表示オプションでサーバ起動

```
hexo server --draft
```

### 下書き記事を公開フォルダへ移動

```
hexo publish introduction
```

### 削除コマンド実施
```
hexo clean
```

### デプロイコマンド実施
```
hexo deploy
```
